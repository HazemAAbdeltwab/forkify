## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/HazemAAbdeltwab/forkify.git
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm start
   ```

4. Build the project for production:
   ```bash
   npm run build
   ```
